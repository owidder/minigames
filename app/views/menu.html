Start Game

Drag each colored circle with a number to the left or the right.

  • To left/right, if it is smaller/bigger than the number of the last circle with the same color
  • If the numbers are the same or you see the color the first time you can drag it to the left or the right
  • The faster you decide the more points you get for a correct decision. BUT BEWARE: If you mess it up, you get the same number of points subtracted
  • 'P:' are your current points and 'R:' the current round, i.e. the number of correct decisions
  • For each wrong decision (are after 5 seconds of not deciding) you loose one life. After 3 lost lifes you're dead
